<h1>Homelab Project</h1>


<h2>Description</h2>
This repository documents my journey building a self-hosted homelab.

The purpose of this project is to gain practical experience in:

- <b>Linux Administration</b> 
- <b>Networking</b> 
- <b>Cloud Infrastructure</b> 
- <b>System Administration</b> 
- <b>Self-hosting</b> 
- <b>Cybersecurity</b> 
- <b>Enterprise Technologies</b> 

Rather than learning solely through tutorials, I wanted to build, maintain, and continuously improve infrastructure that I can use every day.

This repository will continue to evolve as I add new services and technologies.

<h2>Why I Built This</h2>
I created this homelab to move beyond theoretical learning and gain hands-on experience with technologies commonly used in IT infrastructure and cybersecurity. By documenting each stage of the project, I can track my progress, reinforce what I learn, and build a portfolio that reflects practical systems administration and networking skills.
